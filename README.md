# Projeto de Infosustentabilidade das Cidades

## 📘 Descrição
Este projeto tem como objetivo apresentar informações relacionadas à sustentabilidade, meio ambiente e desenvolvimento sustentável das cidades da região do Alto Vale do Itajaí.

As cidades analisadas são:

- Dona Emma
- José Boiteux
- Pouso Redondo

---

## 👨‍💻 Integrantes do Projeto

- Davi
- Lucas
- Tarso

---

## 🌱 Objetivos

- Pesquisar indicadores de sustentabilidade das cidades
- Analisar dados ambientais e sociais
- Promover conscientização sobre desenvolvimento sustentável
- Organizar informações de forma acessível e educativa

---

## 🏙️ Cidades Estudadas

### Dona Emma
Estudo sobre práticas sustentáveis, preservação ambiental e qualidade de vida no município.

### José Boiteux
Análise de aspectos ambientais, recursos naturais e iniciativas sustentáveis locais.

### Pouso Redondo
Levantamento de dados sobre sustentabilidade urbana e impactos ambientais.

---

## 📊 Conteúdos do Projeto

- Dados ambientais
- Informações socioeconômicas
- Indicadores de sustentabilidade
- Gráficos e pesquisas
- Propostas de melhorias sustentáveis

---

## 🛠️ Tecnologias Utilizadas

- Markdown
- GitHub
- Pesquisas online
- Ferramentas de apresentação

---

## 📅 Desenvolvimento

Projeto desenvolvido para fins educacionais e acadêmicos.

---

## 📌 Conclusão

O projeto busca incentivar a conscientização ambiental e destacar a importância da sustentabilidade nas cidades estudadas.
